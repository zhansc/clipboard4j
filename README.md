# 配置多git
## 为避免提交时混淆作者信息，建议在项目级别设置用户
## 在公司以公司gitlab账号为主，所以
git config --global user.name "company"
git config --global user.email "company@ali.com"

## 然后有需要的时候针对个人的github项目级别（每个项目都要重新设置）设置个人GitHub用户
git config user.name "yourname"
git config user.email "youremail@qq.com"
